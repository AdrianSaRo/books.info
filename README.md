**Server:** Vercel adapter

## Run Locally

Clone the project

```bash
  git clone https://github.com/AdrianSaRo/books.info.git
```

Go to the project directory

```bash
  cd books.info
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
